Pemrograman Web

Pada Mata Kuliah ini kami mempelajari beberapa materi terkait pemrograman web diantaranya:

  1. HTML (Hypertext Markup Language)
  2. CSS (Cascading Style Sheets)
  3. JavaScript
  4. PHP
  5. MySQL

Materi pemrograman web yang kami pelajari mencakup beberapa komponen penting dalam pengembangan situs web.

  • HTML (Hypertext Markup Language) digunakan untuk membuat struktur dasar halaman web, seperti judul, paragraf, dan elemen-elemen lainnya.
  • CSS (Cascading Style Sheets) berfungsi untuk mengatur tampilan visual dan tata letak halaman, memberikan gaya dan desain pada elemen HTML.
  • JavaScript adalah bahasa pemrograman yang memungkinkan interaksi dinamis di situs web, seperti validasi form, animasi, atau manipulasi elemen DOM.
  • PHP adalah bahasa server-side yang digunakan untuk menangani logika bisnis dan pengolahan data di server.
  • MySQL adalah sistem manajemen basis data yang digunakan untuk menyimpan dan mengelola data situs web secara efisien, sering kali bekerja sama dengan PHP untuk membangun aplikasi web yang dinamis dan interaktif.
